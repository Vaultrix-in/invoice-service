# 🧾 Vaultrix Invoice Service

Automated billing and document generation for Vaultrix transactions.

## 📄 Features
- PDF generation (using the shared design template).
- Storage and retrieval of historical billing data.


## 🛡️ Security & Quality
This repository is part of the Vaultrix Secure Software Development Lifecycle (SDLC):
- **SonarQube**: Static analysis & Quality Gate.
- **Snyk**: Open-source vulnerability scanning.
- **Trivy**: Container image security scanning.
